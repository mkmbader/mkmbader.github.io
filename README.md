# Maria Bader - Portfolio Website

A responsive AI/ML Engineer portfolio showcasing real projects with modern design and mobile-optimized navigation.

## Quick Start

```bash
python3 -m http.server 8000
```

Open `http://localhost:8000` in your browser.

## Features

- Responsive design (desktop & mobile)
- Project carousel with smooth scrolling  
- Mobile-friendly one-tile-at-a-time navigation
- Interactive project modals
- Real project content integrated

## Design

- **Color Palette**: Purple (#4D2C59), Grey (#9CA3AF), and light accents
- **Typography**: Urbanist font for modern, clean text

## Structure

- `index.html` - Main page
- `styles.css` - All styling (Urbanist font)
- `script.js` - JavaScript functionality and project data
- `image4.jpg` - Hero background

## Technologies

HTML5, CSS3, JavaScript, Google Fonts, Font Awesome
